# CNAPP Dashboard

A React + TypeScript + Tailwind dashboard implementing categories, dynamic widgets (add/remove), and search with local state management via Zustand for the CNAPP assignment. 

## Run locally
- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build`
- Preview production build: `npm run preview`

## Features
- Categories with multiple widgets
- + Add Widget per category or globally
- Remove a widget from a category (via X on the card)
- Manage Widgets panel to uncheck/remove
- Search across all widgets by name/text
- Local persistence (assignment scope)
